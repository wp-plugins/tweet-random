Creates an RSS feed for TweetRandom.com. 

== Description ==

TweetRandom.com is a web app from which you can tweet random stuff from your RSS/Atom feed. This plugin extends it to the Wordpress Platform. More info at [Tweet random](http://suhastech.com/tr).

== Installation ==
1. Unzip and copy the files to your plugins directory (/wp-content/plugins).
1. Activate the plugin.
1. Go to Settings --> TweetRandom --> "Submit your feed to TweetRandom.com"
1. Thats it!
